# Thesis Toolkit

A thesis template for the BSc and MSc Industrial Engineering and Management (IEM) program of the University of Twente.

## Working on Overleaf?

Visit the Overleaf project at:

https://www.overleaf.com/latex/templates/university-of-twente-iem-thesis-template/jprytkvyvmbf

## Working on your PC?

- Install [LaTeX Studio](https://www.latexstudio.net/) (Windows).
- Install [MiKTeX](https://miktex.org/download) (Windows).
- Install [Inkscape](https://inkscape.org/release/inkscape-1.2.2/) (Windows).
- Make sure to add Inkscape to your system PATH during installation.
